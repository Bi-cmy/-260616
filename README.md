# 🤔 你以为的以为 — 概率直觉训练营

一个基于 Streamlit 的交互式概率与统计思维训练平台。通过 15 个精心设计的迷你游戏，训练你对概率、统计陷阱、因果推断和多维思维的直觉。

---

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/Bi-cmy/-260616.git
cd -260616

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动应用
streamlit run app.py
```

启动后在浏览器打开 `http://localhost:8501` 即可进入训练营。

## 游戏目录

| # | 游戏 | 隐藏概念 | 核心直觉 |

## 核心机制

- **金币系统**：完成游戏获得金币奖励
- **成就系统**：达成特定条件解锁隐藏成就
- **雷达图**：9 个维度的直觉能力可视化（独立直觉、条件思维、样本警觉、大数感觉、混淆嗅觉、贝叶斯脑、因果链条、估计智慧、维度直觉）
- **数学细节**：每个游戏附带完整的数学公式推导

## 设计理念

这个训练营的核心理念是：**概率思维不是学出来的，是练出来的。** 每个游戏构造一个具体场景，让你在互动中亲身经历认知偏差，再通过数学细节理解背后的原理。知识和直觉之间隔着一层"做"——这个项目就是那层"做"。

四个章节对应四条从直觉到真相的路径：
1. **赌徒的直觉** → 理解随机性的反直觉本质
2. **真相猎人的调查** → 识破统计数据中的系统性陷阱
3. **因果侦探事务所** → 区分相关与因果的微妙边界
4. **换个维度看世界** → 跳出低维直觉，建立高维思考框架
